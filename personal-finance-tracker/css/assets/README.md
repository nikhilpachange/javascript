# Personal Finance Tracker

## Project Purpose

This Personal Finance Tracker helps users manage their income and expenses. The app provides real-time financial balance information, transaction history, and a summary of the total income and expenses. The app is built using HTML, CSS, and JavaScript.

## Features
- Add income and expense transactions.
- Real-time balance updates.
- Transaction history with editing and deleting features.
- Financial dashboard that displays current balance, total income, and total expenses.
- Data persistence using LocalStorage.
- Responsive design for mobile and desktop.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (Vanilla JS)
- **Libraries**: Chart.js (optional for data visualization)
- **Storage**: LocalStorage (to store transaction data)
